# Web UI 设计

## PC 端布局

```
┌──────────────────────┬─────────────────────────────────────────┐
│ 侧边栏 (280px)        │ 主区域                                   │
│                      │                                          │
│  [项目名称 ▾]         │  顶栏：会话名称 + model badge            │
│  ────────────────    │  ──────────────────────────────────────  │
│  + 新建会话           │                                          │
│  ────────────────    │  聊天视图（消息历史）                      │
│  会话 1              │                                          │
│  会话 2              │                                          │
│  会话 3              │                                          │
│  ...                 │  ──────────────────────────────────────  │
│                      │  输入区                                   │
└──────────────────────┴─────────────────────────────────────────┘
```

## 侧边栏

### 项目切换器
- 顶部显示当前项目名称，点击展开下拉菜单
- 菜单项：所有项目列表 + "管理项目"入口
- 切换项目后，会话列表刷新为该项目下的会话

### 会话列表
每条会话显示：
```
会话名称                          04-15
最后一条消息的预览...
```

- 置顶会话排在顶部，有置顶图标
- 归档会话不在主列表，可通过底部"归档"入口查看
- 右键菜单：重命名、置顶/取消置顶、归档、删除

## 聊天视图

### 消息渲染

| 事件类型 | 渲染方式 |
|---|---|
| message（user） | 右对齐气泡，纯文本 |
| message（assistant） | 左对齐，Markdown 渲染（代码高亮） |
| reasoning | 折叠块"思考过程"，默认收起，可展开 |
| tool_use | 折叠块，显示工具名 + 参数预览 |
| tool_result | 跟在对应 tool_use 后，显示输出预览 |
| file_change | 单行提示，显示文件路径和操作类型 |
| status | 单行状态提示 |
| usage | token 用量小字，显示在回复末尾 |

### 流式输出
- 执行中：assistant 消息逐字追加显示
- 工具调用：先显示 tool_use 块（loading 状态），结果返回后更新为 tool_result
- reasoning：执行中折叠块显示"思考中..."，完成后可展开

### 输入区
- 多行文本框：Enter 发送，Shift+Enter 换行
- 文件上传按钮（附件）
- 运行时选择：model / effort / thinking（下拉/开关）
- 发送按钮（空闲时）/ 取消按钮（执行中）

### 运行状态
- **执行中**：输入框禁用，显示取消按钮，流式输出实时追加
- **队列中**：输入框禁用，显示"等待中 (n)"
- **WS 断线**：状态指示点变灰，3 秒后自动重连，重连成功后增量刷新

## 项目管理页

入口：侧边栏项目切换器 → "管理项目"

功能：
- 项目列表（名称 + 路径）
- 新建项目（输入名称、选择本地路径、填写 system prompt）
- 编辑项目（同上）
- 删除项目（二次确认，提示将删除所有会话）
