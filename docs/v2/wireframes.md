### 任务栏细节

**视图切换**
```
[全部]  [本项目]
```
- 全部：所有项目的 `assignee=human` 未完成任务
- 本项目：仅当前 session 所属项目的任务

**任务卡片（展开态）**
```
┌─────────────────────────────────────────┐
│ □  审核部署计划                           │
│    AI 指示：请检查 docker-compose.yml 里  │
│    的端口配置是否与生产环境一致。           │
│    ─────────────────────────────────── │
│    melody-sync-v2 · 2小时前              │
│    [完成]  [忽略]                        │
└─────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────┐
│ □  审核部署计划          AI · 2h  [✓][✕] │
└─────────────────────────────────────────┘
```

**空状态**
```
┌─────────────────────────────────────────┐
```
